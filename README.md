# Sysdiag Launcher

## Description
AppleSeed な人にはお馴染みの sysdiag コマンドのフロントエンドアプリです。常駐型でマウス操作だけで色々とログが取れます。また、おまけで Wi-Fi 上の tcpdump も取れるようにしています。

MacBook* を使ってる人にはこの tcpdump 取得機能は地味に便利かと信じています。
